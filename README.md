# Run the app

It's currently running at [https://spotify-party.now.sh].

You can also run it locally. Just clone this repo.

Next, run: 

```sh
yarn
```

Then, follow the instructions to run it on web or mobile:

## Mobile

```
npm i -g expo-cli
expo start
```

## Web
```
yarn next
```
